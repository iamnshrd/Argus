# Argus Error Log

Use this as an active list of recurring failure modes. When an error repeats, add the specific case only if it teaches a reusable pattern.

## Rules / Eligibility Errors

- Starting topical or speech-pattern analysis before verifying event eligibility.

- Treating a scheduled TV slot as proof that the resolving speaker will appear live.

- Missing holiday, special-programming, guest-host, prerecorded, replay, archival, or clip-package risk.

- Bonding or treating near-resolution markets as risk-free without fully understanding rules, possible disputes, and maximum downside.

- Case: Rachel Maddow on Memorial Day, May 25, 2026. A public r/msnow programming note about 3-4 hours before air said Maddow would not be live and MS NOW would air special programming. If market rules require live participation, "not live" can send all strikes to NO regardless of prerecorded content.

## Wording Errors

- Confusing "likely topic" with "likely exact mention".

- Assuming logical wording equals natural wording.

- Treating a recent headline as a speech probability.

- Treating the absence of direct pre-event confirmation as a negative signal when partial information is the normal condition of pre-event mention analysis.

- Misreading Николай's phrase "similar to / похожее на" as direct vocabulary transfer or overconfidence. Default interpretation should be comparative format analysis unless he explicitly claims the prior wording transfers.

## Format / Sample Errors

- Ignoring event format.

- Ignoring interviewer incentives.

- Treating a broad transcript hit rate as if it applies to the specific event format.

- Assuming a speaker's old pattern is stable when incentives, office, format, or current events changed.

- Evaluating a long event as one block instead of phase-by-phase.

- Overfitting to a market's old strike set after platforms rotate strikes and the game changes.

## Source Errors

- Using weak sourcing for high-confidence conclusions.

- Overweighting recent headlines.

- Overreacting to fast headlines without reading the full sentence and context.

- Mistaking "people I respect agree" for independent evidence.

- Ignoring that the market may already price the obvious source, promo, or consensus thesis.

## Price / Edge Errors

- Treating "high probability" as "good price".

- Confusing research edge with execution edge; having the right fair value but entering at a bad time or bad price.

- Copying a position without understanding what would invalidate it.

- Treating a full fill as neutral liquidity instead of asking whether the counterparty knows something.

- Ignoring opportunity cost: spending attention on small weak markets while larger cleaner events are coming.
